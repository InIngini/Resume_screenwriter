# Генератор резюме в стиле «Пороки»

Проект представляет собой две веб-страницы: главная страница с готовым резюме и генератор резюме с возможностью редактирования данных и предпросмотра.

## Структура проекта

```
резюме/
├── index.html                # Главная страница с резюме
├── root-style.css            # Стили для главной страницы
│
└── резюме-генератор/         # Папка генератора
    ├── index.html            # Страница генератора
    ├── css/
    │   ├── variables.css     # CSS-переменные
    │   ├── style.css         # Основные стили
    │   └── print.css         # Стили для печати
    └── js/
        ├── main.js           # Данные по умолчанию
        ├── form-handler.js   # Логика формы
        └── pdf-generator.js  # Генерация PDF
```

## Функциональность

### Главная страница
- Отображение готового резюме
- Кнопка перехода в генератор

### Генератор
- Форма для заполнения данных
- Предпросмотр резюме в реальном времени
- Динамические блоки:
  - Навыки
  - Ссылки на портфолио
  - Проекты
  - Причины
  - Кастомные блоки
- Экспорт в PDF

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- html2canvas
- jsPDF

## Установка и запуск

1. Скачать репозиторий
2. Открыть index.html в браузере

## Настройка данных

Данные по умолчанию находятся в файле `резюме-генератор/js/main.js` в объекте DEFAULT_DATA.
